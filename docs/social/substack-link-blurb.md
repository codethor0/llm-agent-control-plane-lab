# Substack link blurb

Companion repo for readers who want a runnable defensive lab:

**llm-agent-control-plane-lab** — a simulated, test-backed reference implementation of an external control plane around tool-connected LLM agents. Deny-by-default policy, broker authorization, provenance checks, human approval for high-impact tools, output filtering, and redacted audit logs. The model can ask; the broker decides.

- Repository: https://github.com/codethor0/llm-agent-control-plane-lab
- Release v0.1.0: https://github.com/codethor0/llm-agent-control-plane-lab/releases/tag/v0.1.0

Local lab use only. Not a production deployment guide.
